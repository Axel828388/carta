# Carta

Una carta interactiva creada con HTML, CSS y JavaScript.

- Abre `index.html` en un navegador para ver la carta.
- Toca el disco para reproducir la canción (`smithereens.mp3`).
- Ingresa el apodo `risita` para abrir la carta.

Para publicar en GitHub Pages:
1. Crea un repositorio llamado `carta` en GitHub (o usa el nombre que prefieras).
2. Añade el remote y haz push (ejemplo más abajo).

SSH example:
```powershell
cd 'C:\Users\axelj\Downloads\carta'
git remote add origin git@github.com:YOUR_GITHUB_USERNAME/carta.git
git push -u origin main
```

HTTPS example:
```powershell
cd 'C:\Users\axelj\Downloads\carta'
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/carta.git
git push -u origin main
```

Nota: Si quieres que el sitio esté en GitHub Pages y el repo sea privado, GitHub Pages desde repositorios privados puede requerir una cuenta/plan que lo permita. Si prefieres, puedo ayudarte a activar Pages una vez que el repo exista.
